# LeetCode Solutions

A collection of LeetCode problem solutions written in Python. Each solution is organized by the primary technique or topic used to solve the problem.

## Repository Structure

The folder layout follows this pattern:

```bash
technique/problem_name.py
```

Example:

```bash
TwoPointer/ReverseString.py
DynamicProgramming/ClimbingStairs.py
Graph/BFS_ShortestPath.py
```

## Adding a New Solution

1. Create a new folder (if needed) for the technique or topic (e.g., `BinarySearch`, `SlidingWindow`, `DynamicProgramming`).
2. Add a Python file named after the problem (e.g., `TwoSum.py`, `ValidParentheses.py`).
3. Write a clean, readable solution and keep helper functions organized.
4. Optionally include a short comment or docstring describing the approach and time/space complexity.

## Running Solutions

Each file is a standalone Python script. Run a solution with:

```bash
python path/to/technique/problem_name.py
```

(If your solution defines a `main()` or test section, it can run sample inputs for quick verification.)
